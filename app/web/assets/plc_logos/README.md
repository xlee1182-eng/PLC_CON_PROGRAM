# PLC Logo PNG Files

Place manufacturer logo PNG files in this folder using the exact names below:

- `opcua.jpg`
- `opcda.jpg`
- `mitsubishi.png`
- `siemens.png`
- `rockwell.png`
- `default.jpg` (optional fallback)

Dashboard lookup path:

- `/static/assets/plc_logos/<file>.(png|jpg)`

If a logo file is missing, the dashboard automatically falls back to an inline SVG badge.
