# PLC Communication Program

## 📌 Project Description
This project is a Python-based PLC communication system.

## 🛠 Tech Stack
- Python 3.x
- Snap7 / OPC
- AsyncIO
- Oracle / PostgreSQL

## 📂 Project Structure
```
src/
  drivers/
  jobs/
  main.py
requirements.txt
```

```bash
git clone https://github.com/your_id/your_repo.git

```

## ▶ Run

```bash
python main.py
```
