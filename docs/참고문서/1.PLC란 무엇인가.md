PLC(Programmable Logic Controller, 프로그래머블 로직 컨트롤러)는 
산업 자동화에서 핵심적인 제어 장치입니다. 교육자료로 바로 활용할 수 있도록 체계적으로 정리해드릴게요.

📘 1. PLC란 무엇인가?

PLC는 공장 설비나 기계를 자동으로 제어하기 위해 만들어진 산업용 컴퓨터입니다.

👉 쉽게 말하면

센서 입력을 받아서

프로그램으로 판단하고

모터, 밸브 등을 제어하는 장치

입니다.

📘 2. PLC가 필요한 이유

기존에는 릴레이(계전기)로 제어했지만 문제점이 많았습니다.

❌ 릴레이 방식 문제

배선 복잡

수정 어려움

유지보수 힘듦

✅ PLC 장점

프로그램으로 제어 (유연성)

배선 감소

유지보수 쉬움

신뢰성 높음

📘 3. PLC 기본 구조

PLC는 크게 5가지로 구성됩니다.

1) CPU (중앙처리장치)

프로그램 실행

연산 및 제어 판단

2) 입력 모듈 (Input Module)

센서 신호 입력

예: 스위치, 센서, 버튼

3) 출력 모듈 (Output Module)

장비 제어

예: 모터, 램프, 밸브

4) 전원부 (Power Supply)

PLC에 전원 공급

5) 통신 모듈 (Communication)

다른 PLC, PC, HMI와 연결

📘 4. PLC 동작 원리 (Scan Cycle)

PLC는 반복적으로 아래 과정을 수행합니다.

🔄 스캔 사이클

입력 상태 읽기

프로그램 실행

출력 갱신

👉 이 과정을 매우 빠르게 반복 (ms 단위)

📘 5. PLC 프로그램 방식

PLC는 주로 다음과 같은 언어를 사용합니다.

1) 래더 다이어그램 (Ladder Diagram)

가장 많이 사용

전기 회로 형태

[ 스위치 ] ----( 모터 )
2) 구조적 텍스트 (Structured Text)

일반 프로그래밍과 유사 (Python 느낌)

3) FBD (Function Block Diagram)

블록 연결 방식

📘 10. PLC vs 일반 컴퓨터
구분	PLC	PC
목적	제어	범용
안정성	매우 높음	보통
실시간성	뛰어남	제한적
환경	산업용	사무용
📘 핵심 정리

✔ PLC는 산업 자동화를 위한 컴퓨터
✔ 입력 → 연산 → 출력 구조
✔ 빠른 반복(스캔)으로 제어 수행
✔ 래더 언어가 가장 많이 사용됨
